# 新手版操作手冊

這份文件給只想使用學習器，或需要最基本題庫管理操作的人。

## 1. 先準備什麼

- 你需要有這個專案的完整檔案。
- 你需要一個瀏覽器。
- 如果要在自己電腦上開啟頁面，建議已安裝 Python，這樣可以快速啟動靜態伺服器。

## 2. 如何啟動學習器

在專案根目錄執行：

```bash
python -m http.server 8000
```

然後在瀏覽器開啟：

- http://localhost:8000/index.html

不要直接雙擊 HTML 檔，否則頁面可能讀不到 JSON 題庫。

## 3. 首頁功能說明

- 開始練習：從完整題庫開始
- 錯題重練：只練目前在錯題庫中的題目
- 學習進度：查看統計與匯出答題紀錄
- 設定：調整題目順序、自動下一題與重設本機資料
- 題庫管理：進入圖片候選圖挑選頁

## 4. 練習規則

- 每題顯示一張圖片與四個英文選項。
- 3 秒後會出現第一層提示。
- 10 秒後會自動顯示答案。
- 你也可以按「提前顯示答案」。
- 如果已經揭答，後續作答會被記錄成看答案後作答。

## 5. 如何查看與匯出紀錄

1. 到「學習進度」頁。
2. 點擊「匯出 JSON 紀錄」或「匯出 CSV 紀錄」。
3. 瀏覽器會下載紀錄檔。

匯出的資料包含：

- 題目 ID
- 中文名稱欄位
- 正確答案與你的答案
- 作答時間
- 提示等級
- 是否已揭答
- 是否為手動揭答
- 分類與圖片路徑

## 6. 如何重設本機資料

1. 到「設定」頁。
2. 點擊「重設本機資料」。
3. 確認後，瀏覽器中的作答紀錄、統計與錯題庫會被清空。

注意：

- 這只會清除你的本機瀏覽器資料。
- 不會修改專案中的正式題庫檔案。

## 7. 如何使用題庫管理頁

如果你只是要幫忙挑圖，不負責正式套用，照下面做即可。

### 先建立管理頁資料

```bash
python tools/question_bank_manager.py manifest
python -m http.server 8000
```

開啟：

- http://localhost:8000/manager.html

### 在管理頁中做什麼

1. 查看目前單字、分類與搜尋字串。
2. 填入中文名稱。
3. 在候選圖中點選最適合的一張。
4. 重複處理下一題。
5. 完成後點「匯出選擇檔」。

### 你需要知道的限制

- 有些題目目前不足四張候選圖，空白格是正常情況。
- 你的管理頁選擇會暫存到瀏覽器 localStorage。
- 匯出後只會得到 JSON 檔，不會自動寫回正式題庫。

正式套用 selection 的步驟請交給會操作維護流程的人，或直接看 [developer-WI.md](developer-WI.md)。

## 8. 常見問題

### 頁面打不開或顯示讀取失敗

通常是因為你直接開 HTML，而不是透過 localhost。

請改成：

```bash
python -m http.server 8000
```

### 管理頁沒有題目

通常是因為還沒執行 manifest。

請先執行：

```bash
python tools/question_bank_manager.py manifest
```

### 匯出按鈕沒反應

請先確認瀏覽器沒有擋下載，並且你真的有做過作答或已經選過圖片。
